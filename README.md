# BUBU_GO

## 用途
每次有某些關鍵字都要去特定網站搜尋很麻煩嗎？

現在就馬上來自定常用的搜尋網站

## 安裝連結
[BUBU_GO](https://goo.gl/7xE2WM)

## Example 1
以instagram為例，有時候在ＦＢ看到貼文，

但是只有附上來源的IG帳號

要在跳到IG搜尋又覺得很麻煩，

這時候只要反白就可以了！！

![example1](https://github.com/JerryLin0920/chrome_extensions_E/blob/master/readmeImage/image2.png)

## Example 2
在蝦皮逛的時候看到便宜的東西，還在懷疑到底是不是淘寶就有的東西

這時候只要反白就可以了！！
